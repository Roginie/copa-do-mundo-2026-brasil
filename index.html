Como funciona a análise

1. Índice de Poder

Cada time recebe uma nota de 0 a 100 com base na força do elenco, investimento, desempenho recente e tradição na Copa do Brasil.

2. Probabilidade de Título

A chance de cada time é proporcional ao seu índice elevado a uma potência (favorece os mais fortes), normalizada para somar 100% entre os candidatos.

3. Confronto Direto

Num jogo entre A e B, a chance de vitória é calculada comparando os índices dos dois (modelo parecido com o de força relativa / Elo).

⚠️ Importante

É um modelo estatístico ilustrativo. Futebol tem zebra, lesão, momento e fator casa. Os números mostram tendência, não certeza.
